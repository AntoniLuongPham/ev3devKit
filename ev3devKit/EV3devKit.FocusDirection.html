Description:

public enum FocusDirection

Specifies the direction to use for focusing the next widget.


Namespace: EV3devKit
Package: ev3devKit

Content:

Enum values:

  • UP - Focus the next widget above the current widget.
  • DOWN - Focus the next widget below the current widget.
  • LEFT - Focus the next widget to the left of the current widget.
  • RIGHT - Focus the next widget to the right of the current widget.
