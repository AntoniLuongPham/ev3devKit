polarity

Description:

public MotorPolarity polarity { set; get; }

Gets and sets the polarity of the motor.

This can be used to invert the positive and negative directions. For example, if you have two continuous rotation servos that are used for driving, you can invert the polarity of the left motor so that a positive position causes both servos to drive forwards.
