GtkScreen

Object Hierarchy:

Description:

public class GtkScreen : Screen

A Screen that can be embedded in a Gtk application.

GtkScreens can be linked in a master-slave type connection so that the UI on the master screen can be viewed simultaneously on the slave screens. This lets you view your UI on multiple resolutions and color depths at the same time.
