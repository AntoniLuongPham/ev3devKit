public enum InputCapability

Capabilities of an input device.


Namespace: Ev3devKit.Devices
Package: ev3devkit

Content:

Enum values:

  • KEY - Key and button press events.
  • RELATIVE_AXIS - Relative axis events, such as mouse movement.
  • ABSOLUTE_AXIS - Absolute axis events, such as joystick position.
  • MISC - Uncategorized events.
  • SWITCH - On/off events.
  • LED - Control LEDs.
  • SOUND - Control sound devices.
  • REPEAT - Repeat events.
  • FORCE_FEEDBACK - Control force feedback.
  • POWER - Power events
  • FORCE_FEEDBACK_STATUS - Force feedback status events.
