min_pulse_sp

Description:

public int min_pulse_sp { set; get; }

Gets and sets the calibration value for the minimum value.

For example, on a 180 degree servo, 500 microseconds should be the drive the motor to -90 degrees. If the actual rotation is not -90 degrees, this value can be adjusted to correct the error.
